## 🚀 Getting Started

### View Online
Simply visit the [live site](https://omarmushtaq-create.github.io/MyWebsite/) to see the portfolio in action.

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/omarmushtaq-create/MyWebsite.git
   cd MyWebsite
   ```

2. **Open in browser:**
   - Double-click `index.html`, or
   - Use a local server (recommended for better performance):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     http-server
     ```

3. **View in browser:**
   - Open `http://localhost:8000` (or your server's URL)

## 🚀 Deployment

This site is automatically deployed to GitHub Pages using the `gh-pages` branch. To deploy your own version:

1. Enable GitHub Pages in repository settings
2. Select `main` branch as the source
3. Your site will be live at `https://[username].github.io/MyWebsite/`

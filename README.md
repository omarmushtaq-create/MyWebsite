## 🚀 Getting Started

### View Online
Visit the [live site](https://omarmushtaq-create.github.io/MyWebsite/)

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/omarmushtaq-create/MyWebsite.git
   cd MyWebsite
   ```

2. **Start a local server:**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (if installed)
   npx http-server
   ```

3. **Open in browser:**
   - Navigate to `http://localhost:8000`

## 📝 How It Works

### Dynamic Navigation
All navigation menus and links are rendered from the `pageContent.navigation` object in `script.js`. Add new sections by updating the object—no HTML changes needed.

### Build Console
The interactive "What I Build With" section dynamically loads skill profiles. Switch between Python, Web, Homelab, and Virtualization to see matching projects.

### Quick Launch
A searchable command palette (⌘K / Ctrl+K or /). Filter all projects, certifications, and pages with natural language matching.

### Parallax & Animations
- Scroll reveal effects for smooth content unveiling
- Parallax sections for depth
- Page transition animations between routes
- Reduced motion support for accessibility

## 🎨 Customization

To adapt this portfolio for your own use:

1. **Update content** in `script.js`:
   - `pageContent.navigation` - Add/remove pages
   - `pageContent.heroStats` - Change stats
   - `pageContent.skills` - Update skill list
   - `pageContent.consoleProfiles` - Modify skill profiles
   - `pageContent.quickLaunch` - Add searchable items

2. **Modify styling** in `styles.css`:
   - Colors, fonts, spacing
   - Responsive breakpoints
   - Animation timing

3. **Update HTML** in individual pages:
   - Project details
   - Certification information
   - Contact methods

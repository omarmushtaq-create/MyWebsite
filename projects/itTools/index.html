IT Tools

This project uses IT-Tools as a self-hosted collection of utilities for development and technical work. I deployed it on my Docker server, configured networking and port mapping, and made sure the service could run reliably in my existing setup. The site gives me quick access to tools for tasks like JSON formatting, encoding and decoding, data conversion, and other everyday IT needs. This project helped me practice Docker deployment, container management, and building useful services that I can host and control myself.

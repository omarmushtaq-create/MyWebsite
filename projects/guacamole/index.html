Guacamole

This project uses Apache Guacamole to provide browser-based remote access to my virtual machines. I configured the server, connected it to my Proxmox lab, and tested remote sessions through the web interface. The project helped me understand remote desktop access, service configuration, and how to manage lab systems more conveniently.
